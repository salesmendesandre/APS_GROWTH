# APS-GROWTH III: Automatización y Autonomía

## Descripción General
Para el curso **2026-2027**, la meta es cerrar el ciclo tecnológico y ecológico, pasando de la observación a la **acción automatizada**. La tercera fase se enfoca en el diseño de un **sistema de riego inteligente** capaz de tomar decisiones autónomas, pero siempre manteniendo un enfoque centrado en la educación y la inclusión del usuario.

## Empoderamiento y Transparencia
El mayor reto de esta fase no es solo regar de forma automática, sino evitar que la tecnología se convierta en una "caja negra". El objetivo es que los usuarios comprendan *por qué* el sistema decide regar. Para ello, el alumnado ha rediseñado la aplicación para que explique, mediante interfaces muy accesibles, las decisiones que toma la máquina, reforzando así el aprendizaje sobre el uso eficiente del agua y el estrés hídrico de las plantas.

## Tecnología Desarrollada
- **Actuadores Inteligentes:** Integración de electroválvulas y caudalímetros para gestionar físicamente el flujo de agua en diferentes zonas del huerto.
- **Algoritmos Predictivos:** Se establecieron umbrales de estrés hídrico rigurosos con el equipo de Botánica para activar el riego solo cuando las plantas realmente lo necesitan, optimizando cada gota de agua.
- **Rediseño Arquitectónico:** La aplicación fue renovada internamente para soportar el procesamiento de datos históricos, manteniendo una interfaz intuitiva que transparenta el proceso de decisión al usuario.

## Impacto
Con esta fase se logra una verdadera **gestión sostenible del agua** y una autonomía técnica completa. El sistema empodera a los usuarios, quienes supervisan a un asistente automatizado, manteniéndolos involucrados en un proceso continuo de aprendizaje ecológico e inclusión tecnológica.
